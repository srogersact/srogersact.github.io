Sentences about Earth Changes That Use "So" and "Because"

You can use so and because to show cause and effect when you join clauses.

"So" is used after the cause and before the effect.
CAUSE, so EFFECT
There has to be a comma before "so."

"Because" is used after the effect and before the cause.
EFFECT because CAUSE
There should be no comma before it.
