Compound Subjects

Compound subjects are very easy. The simplest examples are sometimes so easy that it is tempting to skip reviewing them. However, if we start with very easy examples we can build up to very complicated examples. Then you will be glad you know just what to do.

 A-1   Analysis Example 1 Title

Analysis Example 1 Introduction

Djoser and Snefru ruled Egypt during the Old Kingdom.

This sentence is hard because it has names that are unusual, so let's talk about that.

What to do with names that are hard to pronounce:
  • Don't skip them.
  • Make up a nickname. If you can, think of a name that forms a vivid picture in your mind. The nickname for "Djoser" could be "Dozer" (as in bulldozer). That's easy to imagine. The nickname for "Snefru" could be "Sniff You," which makes me think of my dog.
  • If it is a name you will only be using while working on this website, a nickname will work great and will actually make the ideas easy to remember.
  • If it is a name you keep seeing elsewhere or need to pronounce in public, you'll need to put in the effort to find out how to pronounce it. If you have time, look up the pronunciation online, find it in a dictionary, or ask an adult who knows ancient history.

More Analysis Example 1 Explanation

DjoserNickname "Dozer" ruled Egypt during the Old Kingdom. SnefruNickname "Sniff You" ruled Egypt during the Old Kingdom.
